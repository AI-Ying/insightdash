# Proposal: 产线数据分析 & 历史数据分析

## 1. 背景

已实现质量大屏 MVP（良品率、告警、排名），用户需要增强：
1. **产线数据分析**：支持车间→产线→设备下钻
2. **历史数据分析**：支持按班次/日/周/月查看历史趋势

---

## 2. Scope

### In Scope
- 产线三级下钻（工厂 → 车间 → 产线 → 设备）
- 时间选择器（今日/本周/本月/自定义）
- 班次统计（白班/夜班对比）
- 历史趋势图（日/周/月维度）

### Out of Scope
- 实时 MQTT 接入
- 微信告警通知
- PDF 报告导出

---

## 3. 功能详细设计

### 3.1 产线下钻

#### 交互流程
```
全厂视图 → 点击车间卡片 → 车间视图 → 点击产线行 → 产线视图 → 点击设备行 → 设备详情
```

#### URL 设计
| 视图 | URL 参数 |
|------|---------|
| 全厂 | `/quality` |
| 车间 | `/quality?workshop=压铸车间` |
| 产线 | `/quality?workshop=压铸车间&line=压铸岛1` |
| 设备 | `/quality?workshop=压铸车间&line=压铸岛1&device=压铸机01` |

#### UI 组件
1. **层级导航栏**：显示当前位置（可点击返回上级）
2. **下钻卡片列表**：支持点击进入下级视图
3. **设备详情面板**：侧边滑出，显示单设备详细参数

---

### 3.2 历史数据分析

#### 时间维度
| 维度 | 说明 | 聚合方式 |
|------|------|---------|
| 班次 | 白班(08:00-20:00) / 夜班(20:00-08:00) | 按班次求和 |
| 日 | 每天 00:00-23:59 | 按日求和 |
| 周 | 周一至周日 | 按周求和 |
| 月 | 每月1-30/31 | 按月求和 |

#### 时间选择器
- 快捷选项：今日 / 本周 / 本月 / 上月
- 自定义：日期范围选择器

#### 图表类型
| 图表 | 用途 |
|------|------|
| 折线图 | 趋势展示（日/周/月） |
| 柱状图 | 班次对比（白班 vs 夜班） |
| 表格 | 明细数据 |

## 4. 技术方案

### 4.1 数据层扩展

```typescript
// quality-parser.ts 新增函数
aggregateByShift(records)     // 按班次聚合
aggregateByDay(records)       // 按日聚合
aggregateByWeek(records)      // 按周聚合
aggregateByMonth(records)     // 按月聚合
getDeviceDetail(records, deviceId)  // 设备详情
```

### 4.2 页面结构

```
/quality          # 质量大屏（下钻首页）
/quality/history  # 历史分析
```

### 4.3 新增组件

| 组件 | 说明 |
|------|------|
| `DrillDownNav.tsx` | 层级导航栏 |
| `DeviceDetailPanel.tsx` | 设备详情侧边栏 |
| `TimeRangeSelector.tsx` | 时间范围选择器 |
